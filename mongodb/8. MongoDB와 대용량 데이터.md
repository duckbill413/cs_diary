## 01. Sharding 활성화하기
- Sharded Cluster 를 활용하여 Collection 의 데이터 분산하기
- MongoDB 의 배포 형태
  - Standalone
  - Replica Set <- HA 보장
  - Shared Cluster <- Scale-Out
- Shared Cluster은 MongoDB 의 확장 솔루션
- 컬렉션 데이터의 분산이 가능
- 운영 및 관리가 까다롭다.
- 성능이 느리거나 기타 기능들에 대한 제한이 있다.
