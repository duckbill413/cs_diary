## In-memory DB로서의 Redis
- Redis의 정의
  - Remote Dictionary Server
  - Storage: 데이터 저장소 (데이터 관점)
  - Database: 전통적인 DBMS의 역할을 수행 (영속성 관점)
  - Middleware: 어플리케이션이 이용할 수 있는 유용한 기능을 제공하는 소프트웨어
- Redis로 할 수 있는 것?
  - 아주 빠른 데이터 저장소로 활용
  - 분산된 서버들간의 커뮤니케이션 (동기화, 작업 분할 등)
  - 내장된 자료구조를 활용한 기능 구현
- In-memory DB
  - 데이터를 디스크에 저장하지 않음.
  - 휘발성인 RAM에 저장
  - 빠른 속도
- 빠른 속도와 휘발성의 절충
  - 용도에 맞게 DB와 Redis를 사용
  - 혼합해서 사용 (Cache)
  - Redis의 영속성 확보 (백업 등)
